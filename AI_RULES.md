# AI Rules

This file outlines the rules for AI agents (like Antigravity) working on this repository.

## Commit & Pull Request Guidelines
- **Prefix**: All commits made by the AI must start with the prefix `[AI]`.
  - Example: `[AI] ロジックのリファクタリング`
- **Language**: All commit messages and Pull Request descriptions must be written in **Japanese**.
- **Identity**: Messages must be clearly identifiable as generated by Antigravity. Check the author or use the `[AI]` prefix.
